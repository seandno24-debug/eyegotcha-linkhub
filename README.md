# 션디노 링크허브

번호 기반 링크허브입니다.

## 현재 구조
- 상단: 브랜드 로고 + 브랜드명 + 슬로건
- 중단: 검색/필터
- 하단: 클릭 가능한 상품 카드 목록

## 데이터 소스
- 공개 허브는 Google Sheets CSV를 읽어 상품 카드를 자동 생성합니다.
- 시트 컬럼 권장값:
  - 번호
  - 상품명
  - 이미지 주소
  - 쿠팡 링크

## 운영 방향
- 쇼츠/릴스에서 본 제품을 번호로 바로 찾게 만들기
- 시트에 상품을 추가하면 허브도 자동 반영되게 하기
- 이미지 주소가 비어 있으면 썸네일 폴백을 사용하기

## 파일 안내
- `app.js` : 시트 CSV 연동, 카드 렌더링, 검색
- `index.html` : 페이지 구조
- `styles.css` : 화면 디자인
- `netlify.toml` : Netlify 공개 배포용 설정
- `netlify/functions/coupang-image.js` : 쿠팡 대표 이미지 추출 함수
